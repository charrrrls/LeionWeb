Place your Google Sans Code font files in this folder.

Suggested file names (woff2 preferred):
  GoogleSansCode-Regular.woff2
  GoogleSansCode-Medium.woff2
  GoogleSansCode-SemiBold.woff2
  GoogleSansCode-Bold.woff2

If you don't own a license for Google Sans / Google Sans Code, use an open alternative (e.g. Source Sans 3, JetBrains Mono, Fira Code) and adjust the @font-face declarations in /css/custom.css accordingly.
